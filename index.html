ToolFlow Privacy Policy

Last updated: July 13, 2026

ToolFlow is a browser extension that helps users automatically identify, organize, and manage tool websites and online resources discovered during browsing.

Data Collected

ToolFlow may collect and store the following data locally in the user's browser:

  • Website URL
  • Domain name
  • Page title
  • Favicon
  • Meta description
  • Open Graph title and description
  • Page category and tags
  • Visit count
  • Last visited time
  • Page stay duration

Data Not Collected

ToolFlow does not collect the following data:

  • Names, email addresses, physical addresses, or identity numbers
  • Passwords, authentication credentials, or security answers
  • Payment information or financial records
  • Health information
  • Private messages, emails, or chat content
  • Text entered into input fields
  • Full webpage body content

Local Storage

ToolFlow stores the user's tool library data locally in the browser using browser extension storage. This allows users to keep their personal tool library available across browser sessions.

AI Analysis

If AI analysis is enabled, ToolFlow may send basic webpage metadata to ToolFlow backend services. This metadata may include URL, domain, title, favicon, meta description, Open Graph metadata, headings, keywords, and schema types. This data is used only to classify websites, generate short descriptions, and recommend related tools.

ToolFlow does not upload webpage body text, input field content, passwords, payment information, private messages, or chat content.

Data Sharing

ToolFlow does not sell user data. ToolFlow does not use collected data for advertising. Data sent to ToolFlow backend services is used only for tool identification, classification, recommendation, service reliability, and abuse prevention.

Data Retention

Tool library data is stored locally in the user's browser unless the user clears it using the extension settings or browser storage controls. Server-side metadata used for AI analysis may be cached to improve service reliability and reduce repeated model requests.

User Control

Users can disable automatic mode, disable AI analysis, export local data, or clear local data from the extension settings.
